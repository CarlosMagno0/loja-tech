# 🛒 Loja Tech

Projeto de site de compras desenvolvido com HTML e CSS.

---

## 🚀 Sobre o Projeto

A Loja Tech é um site fictício de e-commerce voltado para produtos tecnológicos.

O projeto simula uma loja online com múltiplas páginas, permitindo navegação entre produtos, carrinho e contato, seguindo os requisitos propostos em atividade acadêmica.

---

## 📄 Estrutura do Site

- Página inicial com produtos em destaque e notícia  
- Página com listagem de produtos  
- Página de carrinho (simulação em HTML)  
- Página de contato com formulário  

---

## ⚙️ Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## 📌 Funcionalidades

- Navegação entre páginas  
- Exibição de produtos com imagem e descrição  
- Layout organizado em cards  
- Cabeçalho e rodapé em todas as páginas  
- Estrutura responsiva básica
