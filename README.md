# 🛒 Loja Tech

🔗 Deploy do projeto: https://lojatech.netlify.app/

---

## 📌 Objetivo do Projeto

A **Loja Tech** é um site fictício de e-commerce desenvolvido como atividade acadêmica, com o objetivo de praticar habilidades de desenvolvimento web utilizando HTML e CSS.

O sistema simula uma loja online de produtos tecnológicos, permitindo navegação entre páginas e visualização de produtos.

---

## 🌐 Acesso ao Projeto

O projeto está hospedado e pode ser acessado pelo link abaixo:

👉 https://lojatech.netlify.app/

---

## 📄 Estrutura do Site

O site é composto pelas seguintes páginas:

- 🏠 Página inicial (Home) com produtos em destaque e notícias
- 🖱️ Página de produtos com listagem de itens
- 🛒 Página de carrinho (simulação de compra)
- 📞 Página de contato com formulário

---

- CSS3
